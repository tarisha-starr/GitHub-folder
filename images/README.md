## Notes

- `.png` works too — if you use PNGs, update the `image` field in
  `content/posts.json` to match.
- Recommended dimensions for Instagram feed: **1080x1350** (4:5).
- Buffer fetches the image from the GitHub raw URL, so files must be
  committed to `main` for the daily run to find them.
