## The production workflow (Steps 1 to 7)

### Step 1. Concept and title
Pull from `channel-concepts.md`. Each channel has 10 video ideas ready.

### Step 2. Script and storyboard
Use the script template in `scripts/02-polarity-crisis.md` as the gold standard:
- Hook in first 5 seconds
- 4 acts with emotional beats
- 20 scenes broken out
- VO direction markers `[pause]` `[slower]` `[whisper]` `[breath]` for ElevenLabs

### Step 3. Generate character reference sheets
Before any scene generation, create 4-pose reference sheets for every recurring character. Use Nano Banana or GPT Image 2.0.

### Step 4. Generate every scene as a still image
For each of the 20 scenes:
- Use the image prompt from the storyboard
- Append the consistency boilerplate (locked at top of every script file)
- Generate in Nano Banana or GPT Image 2.0

### Step 5. Animate each still
- **Seedance 2.0** for cinematic motion, environmental shots, atmospheric, abstract metaphor
- **Kling AI** for character faces, micro-expressions, body movement, gestures
- **Nano Banana video mode** as a fast/free alternative
- Each clip 3 to 5 seconds. Subtle motion beats wild motion.

### Step 6. Voiceover, music, edit
- **ElevenLabs** for VO. Warm mid-50s female voice. Use the `[pause]` `[slower]` `[whisper]` markers from the script.
- **CapCut** (free) for editing. Drop clips in storyboard order, layer VO on top, add music bed (Epidemic Sound "Tender Hours" or "Quiet Comfort"), always-on subtitles.
- Always end with a teaser for the next episode (end-card thumbnails).

### Step 7. Upload and optimise
Each script file ships with:
- 1 thumbnail brief (for Nano Banana or GPT Image 2.0)
- 10 SEO-optimised title options
- A full YouTube description
- 20 tags

**Posting cadence:** 2 to 3 videos per week, 90 days minimum, study which winners over-perform and double down.

## Style rules (inherited from Tarisha's brand)

- No em-dashes
- Always contractions
- British / NZ spelling
- Warm, direct, raw voice (coach-to-friend)
- Personify the body as "she"
- Pet names: "darling," "my love"
