## Phase 5 — Voiceover (per video)

- [ ] ElevenLabs voice chosen (warm mid-50s female, e.g. Charlotte, Lily) OR your own voice clone
- [ ] Full script pasted with all `[pause] [slower] [whisper] [breath]` markers
- [ ] VO generated
- [ ] Full pass listen-back
- [ ] Bad sections regenerated separately
- [ ] Final VO exported as WAV (48kHz, mono)
- [ ] Saved as `audio/{slug}/{slug}-vo.wav`

---

## Phase 6 — CapCut edit (per video)

- [ ] New CapCut project: 16:9, 1080p, 60fps
- [ ] Import all 20 video clips
- [ ] Clips dragged onto timeline in storyboard order
- [ ] VO dropped on audio track 1, aligned to scenes
- [ ] Music bed on audio track 2 (Epidemic Sound, ducked to -18db under VO)
- [ ] Music drops + swells per CapCut notes in script doc
- [ ] Sound design added (kitchen sounds, footsteps, door close, candle flicker)
- [ ] Subtitles auto-generated from VO, styled warm cream serif + dusty rose highlights
- [ ] Hard cuts vs cross-dissolves per script notes
- [ ] Quick colour pass (warmth +5, contrast +3)
- [ ] Full preview watched
- [ ] Exported as `final/{slug}-final.mp4` (1080p high quality)

---
