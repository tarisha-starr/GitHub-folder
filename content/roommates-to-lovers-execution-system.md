# From Roommates to Lovers Membership: Execution System

The daily action plan that makes $10K/month from the $97 membership inevitable.

---

## Context

- **Current goal:** $10,000/month recurring from the membership. At $97/member, that's roughly 103 active members.
- **Available time:** 2 to 4 hours a day around 1:1 clients, plus Monday (no-clients day, content + recovery).
- **Current stage:** intermediate. Decade of clinical practice, existing audience, products built, quiz funnel built. No recurring engine yet. The leak is delivery (newsletter ships inconsistently) and the buyer list is small.

---

## 1. Goal broken into daily actions that directly move revenue

**The maths.** 103 paying members at $97 = $9,991/month. Assume 8% monthly churn (high estimate, somatic-couples niche tends to retain better). Net needs: ~110 active subs at any time, and ~9 new joins a month just to hold steady, plus growth.

**The conversion chain.** Assume:

- 1.5% of Reel viewers comment for the quiz.
- 80% of commenters get the DM and click the quiz.
- 60% of quiz starters complete.
- 4% of quiz completers buy (mix of $27 Deck and $97 membership; half of buyers take the membership directly).

To net 15 new members a month (covers churn plus growth):

- Need ~30 buyers a month (half take membership).
- Need ~750 quiz completers.
- Need ~1,250 quiz starters.
- Need ~1,560 comments triggered.
- Need ~104,000 Reel views/month, roughly 25K a week.

This is achievable on Reels with consistent posting and two of the five angles hitting hard.

**Daily actions tied to revenue, not to vanity:**

| Daily action | Revenue lever it moves |
|---|---|
| Post 1 Reel | Views → comments → quiz |
| Reply to 100% of comments + DMs | Conversion rate from view to click |
| Send the day's quiz commenters into the DM funnel | Quiz starts |
| Reply to every buyer welcome | Retention, week-one experience |
| 15 min on next month's live teach prep | Membership retention |

## 2. The 20% that produces 80% of results

Run every task through this question: does it move new buyers, retained buyers, or buyer trust?

**The 20% (do these, every week, no exceptions):**

1. **Reels.** Five a week, Monday-batched. The traffic engine.
2. **The newsletter.** Twice a week. The leak Tarisha already named in ME.md. Buyers list gets warmed here.
3. **The live monthly teach.** The thing members are paying for. If this slips, churn spikes.
4. **DM and comment replies.** The free conversion multiplier. Five views with a reply outperform fifty without one.
5. **The quiz funnel.** Check it's running, fix any break inside 24 hours.

**The 80% (cut, defer, or batch):**

- New product ideas. Park until the membership is at $10K.
- Admin (already deprioritised per ME.md).
- 1:1 client outreach (the cap is 15, she is at or near it).
- New platform experiments (no LinkedIn, no Substack, no Threads).
- Aspirational calendar entries that have not shipped twice in a row.

## 3. The simple daily execution plan

Built around Tarisha's actual day from ME.md. Body-first before household. Monday is no-clients content day. Phone down by 9pm.

### Monday (content batch day)

| Block | Task |
|---|---|
| Morning, body-first | Yoga + swim |
| 10:00 to 12:00 | Record all five Reels for the week (one block, one outfit, one set-up) |
| 12:00 to 13:00 | Lunch |
| 13:00 to 14:00 | Write or finish both newsletters for the week |
| 14:00 to 15:00 | Plan + outline next live monthly teach (30 min/week × 4 weeks = ready) |
| 15:00 to 16:00 | Reply to all weekend DMs and comments |
| Evening | Phone down by 9pm |

### Tuesday to Friday (client days)

| Block | Task |
|---|---|
| Morning, body-first | Movement before household |
| Between clients (15 to 30 min) | Post the day's Reel, reply to comments and DMs as they land |
| One slot per day, 30 min | One task from the rotation below |
| Evening | Phone down by 9pm |

**Rotation across Tue/Wed/Thu/Fri (one per day):**

- Tue: send the week's two newsletters (one Tue AM, one Thu AM)
- Wed: send one podcast guest pitch
- Thu: one IG live or members' Q&A admin
- Fri: weekly review (see section 6)

### Saturday and Sunday

Off the business. Family. Body. Rest. (No "catch up on work" Sunday night.)

## 4. Content, outreach, offer improvement (the three loops)

**Content loop (daily):**

- Voice-dump → AI draft → Tarisha rewrites → record → post → comment-prompt CTA.
- Source material: a client moment, a DM that landed, a line from her own writing.
- Time per Reel: 15 min, including upload.

**Outreach loop (weekly):**

- One podcast pitch a week, ten weeks a quarter, 30 hosts total in the pipeline.
- One partnership conversation a week (IG live, newsletter swap, joint teach).
- No cold DMs to strangers.

**Offer improvement loop (monthly):**

- Review the last month's quiz completion rate and the quiz-page conversion rate.
- Read the last month's cancellations (one-line exit survey: "what stopped working?").
- Make one improvement to the offer or the page each month. Not three. One.

## 5. Accountability and feedback loops

**One sheet, ten numbers, updated every Friday in 5 minutes.**

| Number | Target | This week | Trend |
|---|---|---|---|
| Reels posted | 5 | | |
| Reel views (total) | 25,000 | | |
| Quiz starts | 300 | | |
| Quiz completes | 180 | | |
| New $27 buyers | 7 | | |
| New $97 members | 8 | | |
| Membership cancellations | <3 | | |
| Active members | 110+ | | |
| MRR | $10,000+ | | |
| Newsletters sent | 2 | | |

**Two rules on the sheet:**

1. If three weeks in a row miss a target, change the input, not the target.
2. If two weeks in a row hit every target, raise the goal by 20%.

**Public accountability (optional but recommended):** one trusted person (Mark, or a peer practitioner) gets the sheet every Friday. No commentary needed. The act of sending it is the accountability.

## 6. Weekly review system (Friday, 30 minutes)

A four-question review. Same questions every week. Done in writing, in the same notebook.

1. **What worked this week?** (Specifically, what produced a buyer or a near-miss?)
2. **What did not ship that should have?** (Newsletter, Reel, teach prep.)
3. **What is the one thing I'll change next week?** (One. Not five.)
4. **Did body-first happen every morning?** (Per ME.md, the leading indicator of everything else.)

Then look at the sheet. Then close the notebook. Then weekend.

## Output: the daily execution system

**Monday:** batch all five Reels, both newsletters, plan teach, clear DMs.
**Tue to Fri:** body-first, clients, one Reel/day, comments and DMs as they land, one rotation task.
**Friday:** 30-min review, update the sheet.
**Weekend:** off.

**One rule that governs all of it:** does this action move new buyers, retained buyers, or buyer trust? If yes, ship it today. If no, park it.

**One metric that runs the week:** new $97 members this week. Everything else is a means to that end.
